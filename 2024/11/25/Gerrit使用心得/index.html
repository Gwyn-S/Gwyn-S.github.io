Gerrit使用心得

提交代码请参考**GIT**

  • 放弃最近一次的提交
  • 单行显示提交log
  • 暂时不是很清楚用法，如果遇到冲突提交可以试着用一下

git pull –rebase

‍

  • 可以通过cherry pick将已经提交的修改同步到本地，用于已经提了修改到远程仓库而本地已经删除了的情况进行还原

​​

  • 对于Jenkins，在Reply中+1之后，可以在下面看到当前代码的排队状态

​ 点击链接进去可以看到如下界面，这个界面包含了当前构建的各种信息

​​

  再次点击返回到项目，可以看到当前工程构建的排队时间，大约一个小时

​​
